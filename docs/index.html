Text-to-SQL Project

About the Project

This project aims to develop a robust Text-to-SQL system that translates natural language queries into SQL statements. It leverages advanced natural language processing techniques and large language models (LLMs) to ensure accurate query generation.

Key Features

  • Supports complex SQL queries, including joins and nested queries.
  • Optimized for geospatial database functions such as PostGIS.
  • Handles diverse natural language input with high accuracy.

Technologies Used

  • Large Language Models (LLMs)
  • PostgreSQL with PostGIS
  • Python and Natural Language Processing libraries

Project Architecture

Project Goals

  • Enable seamless interaction with databases using natural language.
  • Improve the performance of geospatial query translation.
  • Ensure scalability for real-world applications.
